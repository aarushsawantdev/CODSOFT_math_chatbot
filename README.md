# 🤖 Task 1: AI Math Chatbot

This is a rule-based AI chatbot developed using Python that responds to various mathematical queries and performs simple calculations.

## 🚀 Features
- Responds to math-related queries
- Covers Algebra, Trigonometry, and Calculus
- Performs simple calculations
- Interactive command-line interface

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Install Python
2. Run:
   python chatbot.py
3. Start chatting with the bot
